Что такое HTML5?

HTML5 - это пятая версия языка HTML для создания web-страниц. HTML5 содержит множество новых возможностей для создания интерактивных сайтов и веб-приложений, взаимодействия с формами, работы с различными программными интерфейсами, внедрения в страницы мультимедиа-контента, структурирования документов и т.д.

В курсе мы уже рассматривали возможности HTML 5: сворачиваемые/разворачиваемые блоки, новые элементы форм (color, date, month и др.), блочное, строчное и строчно-блочное отображение элементов, flex-контейнеры.
